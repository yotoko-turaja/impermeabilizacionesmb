Declaración de Accesibilidad

1. Compromiso con la accesibilidad

IBÉRICA DE REVESTIMIENTOS está comprometido con garantizar la accesibilidad digital de nuestro sitio web. Nos esforzamos por diseñar y mantener un sitio que sea usable y accesible para todas las personas, independientemente de sus capacidades o las tecnologías que utilicen.

2. Estándares de accesibilidad

Nuestro sitio web ha sido diseñado siguiendo las directrices de Accesibilidad para el Contenido Web (WCAG) 2.1 nivel AA, establecidas por el World Wide Web Consortium (W3C). Estas directrices proporcionan recomendaciones para hacer la web más accesible a personas con discapacidades.

3. Características de accesibilidad implementadas

  • Estructura semántica HTML adecuada
  • Contraste de colores conforme a estándares WCAG
  • Texto alternativo en todas las imágenes
  • Navegación por teclado totalmente funcional
  • Compatibilidad con lectores de pantalla
  • Tamaños de fuente escalables
  • Enlaces descriptivos y claros
  • Formularios etiquetados correctamente

4. Limitaciones conocidas

Aunque nos esforzamos por mantener la máxima accesibilidad, algunos elementos del sitio pueden no ser completamente accesibles debido a limitaciones técnicas. Estamos trabajando continuamente en mejorar estas áreas.

5. Navegación accesible

Puede navegar por nuestro sitio web utilizando:

  • Ratón o panel táctil
  • Teclado (tecla Tab para navegar, Enter para activar enlaces)
  • Lectores de pantalla (JAWS, NVDA, VoiceOver, etc.)
  • Navegadores actualizados

6. Compatibilidad con navegadores

Nuestro sitio web ha sido probado y es compatible con los navegadores más populares, incluyendo Chrome, Firefox, Safari y Edge, en sus versiones más recientes.

7. Apoyo y comentarios

Si encuentras problemas de accesibilidad o tienes sugerencias para mejorar la accesibilidad de nuestro sitio, por favor contacta con nosotros. Tu feedback es valioso para ayudarnos a mejorar continuamente.
